Приветствуем!

Данное ИЭТР - интерактивное электронное руководство представляет из себя пособие по эксплуатации конфигурации базы данных в 1С. База данных направлена на обработку информации о сотрудниках предприятия и проделанной ими работы. Это решает вопросы учета, планирования и управления на предприятии Рсавтодор. База представленна в виде конкретного модуля для предприятия, где рассматриваются рботы проводимые для поддержания и улучшения состояния автомобильных дорог.

ИЭТР предполагается для работы пользователей, имеющих должностные обязонности администраторов на предприятии.

Использование ИЭТР:

Данное ИЭТР содержит в себе семь разделов. В шапке сверху представленно их перечисление. Для перехода на конкретный раздел необходимо кликнуть мышью на его название.
